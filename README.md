*README - Data Analyzer and Transformer Program*

A simple Python console program to analyze and transform data.

*Features*
1. *Input Data* - Enter a 1D array of numbers
2. *Display Data Summary* - Shows count, min, max, sum, and average
3. *Calculate Factorial* - Uses recursion to calculate factorial
4. *Filter Data by Threshold* - Uses lambda to filter numbers above/below threshold
5. *Sort Data* - Sorts data in ascending/descending order
6. *Display Dataset Statistics* - Shows min, max, sum, and average
7. *Exit Program* - Closes the application

*How to Run*
1. Save the code as `data_analyzer.py`
2. Open terminal/command prompt
3. Run: `python data_analyzer.py`
4. Follow the menu and enter your choice 1-7

*Tech Used*
- *Python 3.x*
- *List & Dictionary* - To store and manage data
- *Loops & Conditionals* - For menu and data processing
- *Recursion* - For factorial calculation
- *Lambda Function* - For data filtering

```
Enter your choice: 1
Enter data: 34 12 56 78 43 21 90
Data stored!
```
